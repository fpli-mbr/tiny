# Tiny

> Tiny npm module

[![npm version](https://badge.fury.io/js/%40taniarascia%2Ftiny.svg)](https://badge.fury.io/js/%40taniarascia%2Ftiny)

## Installation

```bash
npm install @taniarascia/tiny
```

## Usage

```js
const tiny = require('@taniarascia/tiny')

tiny('Do the thing')
// => "Dothething"
```
